# DOM RIA XML Feed - Описание полей

Спецификация: https://dom.ria.com/uk/developer/xml/

## Структура документа

```xml
<?xml version="1.0" encoding="UTF-8"?>
<realties
    xmlns="https://dom.ria.com/xml/xsd/"
    xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="https://dom.ria.com/xml/xsd/
    https://dom.ria.com/xml/xsd/dom.xsd">

    <generation_date>2026-02-25T12:00:00+02:00</generation_date>
    <lang_id>2</lang_id>

    <realty>...</realty>
    <realty>...</realty>
</realties>
```

- `generation_date` — дата генерации, формат ISO 8601: `YYYY-MM-DDTHH:mm:ss+03:00`
- `lang_id` — язык фида: `4` = украинский, `2` = русский

## Базовые параметры

| Тег | Тип | Обязательный | Описание | Допустимые значения |
|-----|-----|:---:|-----------|---------------------|
| `email` | string | * | Email пользователя на DIM.RIA | Должен быть зарегистрирован на DIM.RIA, с правами администратора или прикрепленного сотрудника |
| `phone` | string | * | Телефон пользователя | Формат: `+38 (0XX) XXX-XX-XX`, `(0XX) XXX-XX-XX` или `0XXXXXXXXX`. Один номер |
| `local_realty_id` | integer | * | ID объявления в вашей базе | Не более 10 цифр |
| `advert_type` | string | * | Тип операции | `Продажа` |
| `realty_type` | string | * | Тип недвижимости | `Квартира` |
| `realty_sale_type` | integer | * | Тип объекта | `1` — вторичная, `2` — новостройка. По умолчанию `1` |

## Фото

| Тег | Тип | Обязательный | Описание | Допустимые значения |
|-----|-----|:---:|-----------|---------------------|
| `photos_urls` | контейнер | | Контейнер для фото | Минимум 5 фото |
| `loc` | string | | Ссылка на фото | Внутри `photos_urls`. URL по RFC 2396. Может содержать атрибут `crc` (контрольная сумма) |

## Видео

| Тег | Тип | Обязательный | Описание | Допустимые значения |
|-----|-----|:---:|-----------|---------------------|
| `youtube_link` | string | | Ссылка на YouTube видео | Полная ссылка, например `https://www.youtube.com/watch?v=...` |
| `tiktok_link` | string | | Ссылка на TikTok видео | Полная ссылка, например `https://www.tiktok.com/@.../video/...` |

## Расположение

| Тег | Тип | Обязательный | Описание | Допустимые значения |
|-----|-----|:---:|-----------|---------------------|
| `state` | string | * | Область | Пример: `Одесская`, `Киевская` |
| `city` | string | * | Город | Пример: `Одесса`, `Киев` |
| `district` | string | | Район города | Из базы DIM.RIA |
| `metro` | string | | Станция метро | Из базы DIM.RIA |
| `street` | string | * | Название улицы | Без типа улицы |
| `street_type` | string | * | Тип улицы | `улица`, `переулок`, `проспект`, `бульвар`, `набережная`, `площадь`, `шоссе`, `тупик` и др. |
| `building_number` | string | * | Номер дома | Заглавные буквы на украинском, без пробелов. Пример: `23А`, `27/20`, `4А корпус 9` |
| `show_building_no` | integer | | Выводить номер дома | `1` — да, `0` — нет. По умолчанию `1` |
| `number_not_assigned` | string | | Номер дома еще не присвоен | `да`, `нет`. По умолчанию: `нет` |
| `show_flat_no` | integer | | Выводить номер квартиры | `1` — да, `0` — нет. По умолчанию `1` |
| `newbuild_id` | integer | | ID новостройки на DIM.RIA | |
| `newbuilding_name` | string | | Название ЖК | Из списка на DIM.RIA |
| `flat_number_str` | string | | Номер квартиры | Пример: `23А`, `23/1`, `23А-1` |
| `radius_location` | string | | Показывать зону вместо точных координат | Всегда `да` |
